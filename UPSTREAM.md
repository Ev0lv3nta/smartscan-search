# Происхождение и синхронизация

## Исходный проект

SmartScan Search создан как форк [smartscanapp/smartscan-android](https://github.com/smartscanapp/smartscan-android). Начальная точка разработки:

- commit: [`1209599d70196f63dad3762fc6adee5f9414726d`](https://github.com/smartscanapp/smartscan-android/commit/1209599d70196f63dad3762fc6adee5f9414726d);
- дата: 8 июля 2026 года;
- лицензия: GNU GPL v3.0.

История Git сохранена, а GitHub отображает связь форка с исходным репозиторием. Авторство исходного кода не изменяется.

SmartScan Search является независимым неофициальным проектом. Упоминание SmartScan описывает происхождение кода и не означает одобрение, поддержку или аффилированность с авторами исходного проекта. Перед первым самостоятельным релизом приложение получит отдельные package ID, название и визуальную идентичность; исходные обозначения останутся только там, где они нужны для атрибуции и совместимости.

## ML-зависимость

На начальной стадии приложение использует `com.github.smartscanapp.smartscan-android-lib:smartscan-ml:2.2.0`, опубликованную из [smartscanapp/smartscan-android-lib](https://github.com/smartscanapp/smartscan-android-lib). Планируемое включение исходников SDK в monorepo будет выполнено отдельным pull request с указанием точной ревизии, лицензии и локальных изменений.

## Обновления upstream

Удалённый репозиторий `upstream` указывает на исходный SmartScan. Обновления не сливаются автоматически: каждый перенос проверяется на совместимость с архитектурой поиска, схемами данных и воспроизводимостью сборки. В pull request синхронизации перечисляются перенесённые commits и разрешённые конфликты.
